# EdTech Platform

Современная образовательная платформа для управления учебным процессом с ролями студентов и преподавателей.

## Функционал

### Авторизация
- Регистрация с инвайт-кодами для преподавателей и групп
- Вход в систему с JWT токенами
- Автоматическое обновление токенов
- Ролевая система доступа (студент/преподаватель)

### Профили пользователей
- Профили студентов и преподавателей
- Загрузка аватарок с автоматическим сжатием в AVIF
- Отображение статистики (XP, уровень, кристаллы)
- Управление личными данными

### Статистика и прогресс
- Статистика учеников (ранг, выполненные/невыполненные задания, общий XP)
- График средних оценок по месяцам (Chart.js)
- Лидерборд группы с ранжированием по XP
- Автоматический расчёт и кэширование статистики на бэкенде

### Расписание и уроки
- Просмотр расписания на неделю
- Текущий урок с детальной информацией
- Отслеживание посещаемости
- Выставление оценок за уроки
- Управление уроками для преподавателей

### Система заданий
- Создание заданий преподавателями с дедлайнами
- Лента заданий для студентов
- Отслеживание статусов (выдано, отправлено, на проверке, оценено, просрочено)
- Управление группами и инвайт-кодами

### Магазин
- Каталог товаров для покупки за кристаллы
- Покупка и активация товаров
- История покупок
- Управление покупками преподавателями

### Навигация и интерфейс
- Адаптивный хедер с профилем и статистикой
- Боковая панель с контекстным меню в зависимости от роли
- Профессиональный футер с контактами и навигацией
- Современный дизайн с Bootstrap 5

### Игры для студентов
- Рулетка для развития лудомании

## Технологии

### Frontend
- React 18
- React Router
- Bootstrap 5
- Axios
- Chart.js (графики статистики)

### Backend
- Django 4
- Django REST Framework
- JWT Authentication
- SQLite
- Pillow (обработка изображений)

## API Endpoints (не все)

### Авторизация
- `POST /api/register/` - Регистрация
- `POST /api/login/` - Вход

### Профили
- `GET /api/student/profile/` - Профиль студента
- `GET /api/teacher/profile/` - Профиль преподавателя
- `PATCH /api/student/profile/` - Обновление профиля студента
- `PATCH /api/teacher/profile/` - Обновление профиля преподавателя

### Статистика
- `GET /api/student/statistics/` - Статистика ученика (ранг, задания, XP)
- `GET /api/student/leaderboard/` - Лидерборд группы
- `GET /api/student/monthly-grades/` - Средние оценки по месяцам для графика

### Расписание и уроки
- `GET /api/schedule/` - Расписание на неделю
- `GET /api/lessons/{id}/` - Детали урока
- `GET /api/lessons/{id}/attendance/` - Посещаемость урока
- `POST /api/lessons/{id}/attendance/` - Сохранение посещаемости и оценок

### Задания
- `GET /api/assignments/` - Список заданий
- `POST /api/assignments/` - Создание задания
- `PATCH /api/assignments/{id}/` - Обновление задания

### Магазин
- `GET /api/shop/items/` - Каталог товаров
- `POST /api/shop/buy/{id}/` - Покупка товара
- `GET /api/shop/purchases/` - История покупок
- `POST /api/shop/purchases/{id}/activate/` - Активация покупки

## Основные возможности

### Для студентов
- Просмотр и выполнение заданий с отслеживанием дедлайнов
- Покупка товаров в магазине за кристаллы
- Отслеживание прогресса (XP, уровень, кристаллы)
- Просмотр расписания уроков
- График средних оценок по месяцам
- Лидерборд группы для сравнения с другими учениками
- Управление профилем и аватаркой
- Играть в рулетку на кристалики

### Для преподавателей
- Создание и управление заданиями
- Управление группами студентов
- Генерация инвайт-кодов для групп
- Просмотр и управление покупками студентов
- Просмотр расписания уроков
- Управление посещаемостью и выставление оценок
- Отслеживание статистики учеников

## Установка и запуск

### Backend
```bash
cd back/backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend
```bash
cd EdTechFront
npm install
npm start
```

## Особенности реализации

- **Кэширование статистики:** Средние оценки по месяцам рассчитываются и сохраняются в бэкенде для оптимизации производительности
- **Автоматическое сжатие изображений:** Аватарки автоматически сжимаются в формат AVIF при загрузке
- **JWT авторизация:** Автоматическое обновление access токенов через refresh токены
- **Ролевая система:** Разделение прав доступа между студентами и преподавателями
- **Динамические графики:** Графики статистики с адаптивной шкалой на основе данных
- **Рулетка на кристаллы:** Игра на тест лудомании
